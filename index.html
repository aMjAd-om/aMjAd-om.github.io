RAG-Enhanced PDF Knowledge Base

This project allows users to upload PDFs, storing them in MongoDB. Then, users can chat with the system using Retrieval-Augmented Generation (RAG), where an AI orchestrator delegates queries to specialized agents (table, graph, or text) for accurate responses. It leverages advanced AI to deliver precise, context-aware answers from the uploaded documents.

This project leverages YOLOv8 to train a face-mask detection system, identifying whether individuals are wearing masks in real time.
